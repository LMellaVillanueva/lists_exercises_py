Manipulación de listas en Python


Descripción del proyecto

Resolución de una serie de ejercicios enfocados a la manipulación de listas en Python utilizando SLICING, CONCAT, ASIGNACIÓN POR ÍNDICE y otras operaciones.
El objetivo de este proyecto es reforzar el aprendizaje y mejorar la comprensión de estructuras de datos básicas.


Objetivos de aprendizaje

° Utilizar 'slicing' para extraer properciones de una lista.
° Aplicar 'concatenación' entre listas.
° Manipular elementos mediante índices.
° Invertir, duplicar y extraer elementos de una lista.
° Aplicar buenas prácticas.


Ejercicios

1. El mensaje secreto

Extraer el mensaje "secreto" desde una lista que contiene caracteres individuales usando slicing.

2. Intercambio de posiciones

Intercambiar el primer y último elemento de una lista utilizando solo índices.

3. El sándwich de listas

Unir varias listas para formar una sola lista con un orden específico.

4. Duplicando la lista

Crear una nueva lista cuyo contenido duplique el de la lista original.

5. Extrayendo el centro

Encontrar el elemento central de una lista con longitud impar.

6. Reversa parcial

Invertir únicamente la primera mitad de una lista utilizando slicing.


Ejecución

1. Clonar este repositorio:
    git clone https://github.com/LMellaVillanueva/lists_exercises_py.git

2. Entrar al directorio del proyecto:
    cd ejercicios

3. Ejecutar nombre del archivo Python:
    python lists.py
